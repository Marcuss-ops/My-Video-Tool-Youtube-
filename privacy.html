YouTube API Services

La nostra applicazione utilizza i YouTube API Services. Utilizzando questo strumento, accetti di essere vincolato dai YouTube Terms of Service.

Non raccogliamo, memorizziamo o condividiamo dati personali degli utenti con terze parti. I dati ottenuti tramite le API di YouTube sono utilizzati esclusivamente per caricare i video dell'utente sul proprio canale.

Puoi revocare l'accesso ai tuoi dati in qualsiasi momento tramite la pagina delle impostazioni di sicurezza di Google.

Per ulteriori informazioni, consulta la Google Privacy Policy.
